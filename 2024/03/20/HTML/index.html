HTML

HTML

元素

  • HTML元素以开始标签起始，以结束标签结束
  • 元素内容是开始标签与结束标签之间的内容
  • 空元素在开始标签中进行关闭
  • 元素可拥有属性
  • 元素可以嵌套，必须有始有终，不能交叉嵌套
  • 块(block)元素包含整块区域的内容，浏览器在块元素之间用空白边缘分隔开，浏览器会自动地在块元素的前后添加空行，以新行开始和结束
  • 内联(inline) 元素作用在一小部分内容上，浏览器允许多个内联元素出现在同一行，必须嵌在块元素中

属性

  • 属性可以在元素中添加附加信息
  • 属性一般描述于开始标签
  • 属性总是以名称/值对的形式出现
  • 属性常用引用属性值 i.e. name=”value”
  • 常用属性
    • class
      • 为html元素定义一个或多个类名（classname）(类名从样式文件引入)
      • class=”多个class属性的引用”
    • id
      • 定义元素的唯一id
      • id=”唯一id”
    • style
      • 规定元素的行内样式（inline style）
    • title
      • 描述了元素的额外信息 （作为工具条使用）

标签

头部元素容器 head

  • title
    • 定义文档的标题（必需且唯一）
    • 浏览器工具栏中的标题、页面被添加到收藏夹时的标题、在搜索引擎结果中的页面标题
    • 支持全局属性
  • meta
    • 描述文档的元数据(空元素)
    • 指定字符编码，网页的描述，关键词，文件的最后修改时间，作者及其他元数据
    • 被使用浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 Web 服务调用

定义文档的主体 body

  • h1 to h6
    • 定义从大到小的标题（块）
  • hr
    • 水平线分隔 HTML 页面中的内容（块，空元素）
  • <!— 这是一个注释 —>
    • 注释（块，空元素）
    • 浏览器忽略注释
  • p
    • 定义段落（块）
  • br
    • 插入单个换行（内联，空元素）
  • 文本格式化（内联）
    • em
      • 斜体字强调
    • strong
      • 粗体字强调
  • a
    • 设置超文本链接（内联）
    • 属性 href 指定链接目标的URL
    • 链接 i.e. href=”http://www.google.com“
    • 锚点 i.e. href=”lectures.html”
  • img
    • 在网页中插入图片（内联，空元素）
    • 源属性 src 指定图片的URL链接
    • 属性 alt 为图像定义一串预备的可替换的文本
  • table
    • 定义表格（块）
    • tr
      • 定义表格行
    • th
      • 定义表头
    • td
      • 定义表格单元（内联）
    • caption
      • 定义表格标题
  • 列表
    • ul
      • 定义无序列表（块）
      • li
        • 定义列表项
    • ol
      • 定义有序列表（块）
      • li
        • 定义列表项
    • dl
      • 定义自定义列表（块）
      • dt
        • 定义列表项目
      • dd
        • 定义列表项目的描述
  • blockquote
    • 定义摘自另一个源的块引用（块）
    • 一般嵌套段落元素
  • div
    • 组合其他 HTML 元素的容器（块）
    • 用于文档的区域布局
  • span
    • 文本的容器（内联）
    • 组合文档中的行内元素
  • 字符实体
    • 在网页中表示任何unicode字符可以用&xxx表示
  • code
    • 定义计算机代码文本（内联）
  • pre
    • 定义预格式化的文本（块）
    • 保留空格和换行符
  • form
    • 创建表单（块）
    • 属性 action 定义表单数据提交的目标 URL
    • 属性 method 定义提交数据的 HTTP 方法
    • label
      • 为表单元素添加标签
      • 属性 for 定义标签
    • input
      • 定义输入元素，创建文本输入框、密码框、单选按钮、复选框
      • 属性 type 定义输入框的类型
      • 属性 id 关联label元素
      • 属性 name 标识表单字段
    • select
      • 创建下拉列表
      • option
        • 定义下拉列表中的选项
  • iframe
    • 定义框架，在同一个浏览器窗口中显示不止一个页面
