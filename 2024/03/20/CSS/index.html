CSS

CSS

语法

  • selector {property:value;}
  • 选择器selector，声明property:value;
  • /*注释*/

选择器

  • id选择器
    • #idname {property:value;}
    • 指定标有特定 id 的 HTML 元素特定的样式
    • CSS中选择HTML元素
  • class选择器
    • .classname {property:value;}
    • 描述一组元素的样式，可以在多个元素中使用
    • HTML中选择CSS样式
    • p.classname {property:value;}
    • 指定特定的所有 HTML 元素使用 class

插入样式表

  • 外部样式表
    • 在头部 head 中使用 link 标签（空元素）链接到样式表
    • \
    • 属性 rel 定义当前文档与被链接文档之间的关系（必需）
    • 属性 type 规定被链接文档的 MIME 类型
    • 属性 href 规定外部资源（通常是样式表文件）的位置（URL）
  • 内部样式表
    • 在头部 head 中使用 style 标签定义内部样式表
  • 内联样式
    • 标签内使用 style 样式属性指定CSS声明
  • 多重样式优先级
    • 内联>内部>外部>浏览器默认
    • 内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器
  • 多重样式继承
    • 多个样式应用到同一个元素，可以被继承
    • 不是所有样式都可以被继承
    • 更紧密匹配的规则可以覆盖更通用的规则
    • 按优先级继承，优先级相同新样式覆盖旧样式

分组和嵌套

分组选择器

  • 选择相同样式的元素
  • h1, h2, p{}

嵌套选择器

  • 适用于选择器内部的选择器的样式
  • .mark p{}
    • 为所有class=”mark”元素内的p元素指定一个样式
  • p.mark{}
    • 为所有class=”mark”的p元素指定一个样式

组合选择符

  • 后代选择器
    • 空格分隔
    • 选取某元素的后代元素
  • 子元素选择器
    • > 号分隔
    • 选择作为某元素直接/一级子元素的元素
  • 相邻兄弟选择器
    • + 号分隔
    • 选择紧接在另一元素后的元素，且二者有相同父元素
  • 一般兄弟选择器
    • ~ 号分隔
    • 选取所有指定元素之后的兄弟元素

伪类

  • pseudo-classes
  • 添加一些选择器的特殊效果
  • 以选中元素为基准点，此元素的一些状态或属性
  • 伪类选择元素基于的是当前元素处于的状态，或者说元素当前所具有的特性
  • 语法
    • 选择器与伪类
      • selector:pseudo-class {property:value;}
    • CSS类与伪类
    • selector.class:pseudo-class {property:value;}
  • anchor 伪类
    • 以不同方式显示链接的不同状态
    • a:link{} 未访问
    • a:visited{} 已访问
    • a:hover{} 鼠标划过链接
    • a:active{} 已选中
  • first-child 伪类
    • 选择父元素的第一个子元素
  • lang 伪类
    • 为不同的语言定义特殊的规则

伪元素

  • 添加一些选择器的特殊效果
  • 伪元素是对元素中的特定内容进行操作
  • 设计伪元素的目的就是去选取诸如元素内容第一个字（母）、第一行，选取某些内容前面或后面这种普通的选择器无法完成的工作
  • 语法
    • 选择器与伪元素
      • selector:pseudo-element {property:value;}
    • CSS类与伪元素
      • selector.class:pseudo-element {property:value;}
  • 可以结合多个伪元素来使用
  • first-line 伪元素
    • 向文本的首行设置特殊样式
    • 只能用于块级元素
  • first-letter 伪元素
    • 向文本的首字母设置特殊样式
    • 只能用于块级元素
  • before 伪元素
    • 在元素的内容前面插入新内容
  • after 伪元素
    • 在元素的内容之后插入新内容

CSS属性

背景

  • 定义HTML元素的背景
  • background-color
    • 定义元素的背景颜色
    • 颜色值为十六进制、RGB或颜色名称
  • background-image
    • 描述元素的背景图像
    • 值为url(图像url)
  • background-repeat
    • 设置背景图像水平或垂直平铺或不平铺
    • repeat-x, repeat-y, no-repeat
  • background-attachment
    • 设置背景图像是否固定或者随着页面的其余部分滚动
  • background-position
    • 设置背景图像的起始位置
  • background
    • 简写，属性按以上顺序用空格分隔

文本

  • color
    • 文字颜色
  • text-align
    • 水平对齐方式
    • 居中、左对齐、右对齐、两端对齐
    • center, left, right, justify
  • text-decoration
    • 修饰
    • 无（删除链接的下划线）、上划线、中划线、下划线
    • none, overline, line-through, underline
  • text-transform
    • 文本转换
    • 全大写、全小写、单词首字母大写
    • uppercase, lowercase, capitalize
  • text-indent
    • 文本缩进
    • 像素、em、百分比
    • px, em, %

字体

  • font-family
    • 文本的字体系列
    • 字体系列的名称超过一个字，它必须用引号
  • font-style
    • 字体样式
    • 正常、斜体、倾斜
    • normal, italic, oblique
  • font-size
    • 字体大小
    • 像素、em、百分比
    • px, em, %
  • font-variant
    • 设置小型大写字母的字体显示文本
  • font-weight
    • 字体的粗细
    • 默认、粗体、更粗体、细
    • normal, bold, bolder, lighter

尺寸

  • 控制元素的高度和宽度，增加行间距
  • height
    • 设置元素的高度
  • width
    • 设置元素的宽度
  • max/min-height/width
    • 元素的最大/小高/宽度
  • line-height
    • 设置行高

显示

  • 设置元素显示方式
  • display:none vs. visibility:hidden
    • 前者隐藏的元素不会占用任何空间，后者隐藏的元素仍占用与未隐藏之前一样的空间
  • display:inline/block
    • 更改显示方式为内联/块元素

定位

  • 指定元素的定位类型
  • position
    • static
      • 默认值，没有定位，不受top等影响
    • fixed
      • 位置相对于浏览器窗口是固定位置
      • 窗口滚动也不会移动
    • relative
      • 相对其正常位置定位
      • 相对定位元素经常被用来作为绝对定位元素的容器块
    • absolute
      • 位置相对于最近的已定位父元素或\
      • 定位的元素和其他元素重叠
    • sticky
      • 基于用户的滚动位置来定位
      • 元素定位表现为在跨越特定阈值前为相对定位，之后为固定定位
      • 必须指定至少一个上右下左才能使粘性定位生效
  • 重叠的元素
    • 属性 z-index 指定元素的堆叠顺序，大的显示在最前面
    • 未指定 z-index 的两个定位元素重叠，最后定位在HTML代码中的元素将被显示在最前面

overflow

  • 控制内容溢出元素框时在对应的元素区间内添加滚动条
  • visible
    • 默认值。内容不会被修剪，会呈现在元素框之外
  • hidden
    • 内容会被修剪，并且其余内容是不可见的
  • scroll
    • 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容
  • auto
    • 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容
  • inherit
    • 规定应该从父元素继承 overflow 属性的值

浮动

  • 设置元素向左或向右移动，其周围的元素也会重新排列
  • 如果有空间的话，相邻的浮动元素将彼此相邻
  • clear
    • 指定元素两侧不能出现浮动元素，使元素不会重新排列
    • clear:both
  • float
    • left, right
    • none, inherit

对齐

  • 元素居中对齐
    • margin:auto;
    • 未设置width则不起作用
  • 文本居中对齐
    • text-align:center;
  • 图片居中对齐
    • margin: auto;并将图片放到块元素中
  • 左右对齐
    • 定位方式
      • position: absolute;
    • 浮动方式
      • float: right;
  • 垂直居中对齐
    • padding: 70px 0;
    • line-height: 1.5;
    • position: absolute; transform: translate(-50%, -50%);

盒子模型

  • 所有HTML元素可以看作盒子，CSS封装周围的HTML元素
  • 从外到内
    • Margin(外边距) 清除边框外的区域，外边距是透明的
    • Border(边框) 围绕在内边距和内容外的边框
    • Padding(内边距) 清除内容周围的区域，内边距是透明的
    • Content(内容) 盒子的内容，显示文本和图像
  • 宽度和高度
    • width, height
    • 是内容的宽度和高度
  • 总元素的宽度=宽度+左填充+右填充+左边框+右边框+左边距+右边距
  • 总元素的高度=高度+顶部填充+底部填充+上边框+下边框+上边距+下边距

边框

  • border-width
    • 简写属性，为元素的所有边框设置宽度，或者单独地为各边边框设置宽度
  • border-style
    • 必需属性，设置元素所有边框的样式，或者单独地为各边设置边框样式
  • border-color
    • 简写属性，设置元素的所有边框中可见部分的颜色，或为 4 个边分别设置颜色
  • 简写属性单独地为各边设置的顺序
    • 上->右->下->左

轮廓

  • outline-color
    • 颜色
  • outline-style
    • 样式
  • outline-width
    • 宽度
  • outline 是不占空间的

外边距

  • margin
    • 简写属性，在一个声明中设置所有外边距属性
    • 外边距是完全透明的
    • 只能设置外边距的宽度
      • auto, length, %
  • margin-bottom/left/right/top

填充

  • padding
    • 简写属性，在一个声明中设置所有填充属性
    • 填充元素的背景颜色
    • 只能设置填充的宽度
  • padding-bottom/left/right/top
