JavaScript

JavaScript

入门

简介&用法

  • JavaScript是一种轻量级的、可插入HTML页面的脚本语言
  • HTML中的JS脚本代码必须位于 script 元素内
    • 放在 body 元素内
    • 放在 head 元素内
    • 外部 JS 文件
      • script 元素中 src 属性设置JS文件位置

输出

  • 无任何打印或输出的函数
  • 显示数据方式
    • 弹出警告框 window.alert()
    • 将内容写到HTML文档中 document.write()
    • 写入到HTML元素 innerHTML
    • 写入到浏览器的控制台 console.log()

语法

  • 字面量
    • 固定值
    • 数字字面量如3.14，字符串字面量如”qq”和’qq’
    • 表达式字面量如5+6
    • 数组字面量如[1,2,3]，对象字面量如{key:value}
    • 函数字面量如function myfunc(a,b){return a+b;}
  • 变量
    • 存储数据值
    • 使用关键词 var 定义变量，使用 = 为变量赋值
  • 运算符
    • 赋值、算术和位运算符
    • 条件、比较和逻辑运算符
  • 语句
    • 分号分隔
  • 关键字
  • 注释
    • //注释
  • 数据类型
  • 函数
    • 引用函数=调用函数(执行函数内语句)
  • 对字母大小写敏感
  • 字符集采用Unicode

基础

语句

  • 可选分号 ; 分隔
  • 按顺序依次执行代码每条语句
  • 花括号组合代码块
  • 忽略多余空格
  • 可以在文本字符串中使用反斜杠 \ 对代码行进行换行

注释

  • 单行 //
  • 多行 /**/

变量

  • 存储信息的容器
  • 声明（创建）变量
    • 关键词 var
    • 允许一条语句多个变量
    • 未使用值来进行声明的变量其值为 undefined
    • 重新声明变量，变量的值不会丢失

数据类型

  • 支持动态类型，相同变量可用作不同的类型
  • typeof 运算符查看变量的数据类型
  • 基本类型
    • 字符串
    • 数字
    • 布尔
      • true / false
    • Undefined 和 Null
      • undefined表示不含有值，表示没有设置值的变量
      • null表示清空变量，表示一个空对象引用，主动释放一个变量引用的对象
      • null 和 undefined 值相等但类型不等
  • 引用类型
    • 数组
      • 同类型数据集合
      • var array=[];
    • 对象
      • 键值对数据集合
      • var object={key:value, key:value};
      • 两种寻址方式
        • object.key
        • object[“key”]
    • 函数
    • 正则
    • 日期
  • 声明新变量时，可以用关键字 new 来声明类型
  • typeof 操作符检测变量的数据类型
  • constructor 属性返回所有 JavaScript 变量的构造函数
  • 类型转换
    • 显式
      • 构造函数
      • 全局方法
    • 隐式
      • 运算符
      • 输出

对象

  • 拥有属性和方法的数据
  • 一切皆对象
  • 对象是变量的容器
  • 对象是属性和方法的容器
  • 对象属性
    • 键值对创建
      • key:value
    • 两种访问方式
      • object.key
      • object[“key”]
  • 对象方法
    • 创建
      • methodName : function() {//code}
    • 定义函数并作为对象属性存储
    • 对象方法通过 () 调用
    • 不加 () 访问方法会作为定义函数的字符串返回

函数

  • 函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块
  • 函数语法
    • function functionName(){//code}
  • 调用带参数的函数
    • 传入基本类型是传值
      • 不会修改显式参数的初始值
    • 传入引用类型是传址
      • 修改对象属性可作用于函数外部
      • 如果替换整个参数则不会影响原始值
  • 调用带返回值的函数
    • 关键字 return
    • 执行 return 语句，函数返回(值可选)
  • arguments 函数内置对象
    • 包含函数调用的参数数组
  • 函数调用
    • 全局变量函数调用
    • 作为对象方法调用
    • 使用构造函数调用函数
      • 创建一个新的对象，继承构造函数的属性和方法
    • 作为函数方法调用函数
      • call(对象本身, 参数1, 参数2)
      • apply(对象本身, 参数数组)

类

  • 语法
    • class ClassName{constructor(){//code}}
    • 类中包含构造函数方法，用于创建和初始化 class 的对象
    • 不定义构造方法则自动添加空构造方法
  • 使用类
    • new 关键字创建对象，自动调用构造函数方法 constructor()
    • 类表达式
  • 类的方法
    • 添加一个构造函数方法
    • 添加任意数量的方法
  • 类继承
    • extends 关键字
      • class Dog extends Animal{//code}
    • super()方法引用基类的构造方法
  • getter and setter
    • 获取和设置值，都需要在严格模式下执行
    • get and set 关键字创建方法
    • getter 是一个方法，但是获取属性值时无需使用括号
    • setter 是一个方法，但是使用与设置属性值时相同的语法
  • 类声明不会提升
  • 静态方法
    • static 关键字修饰的方法
    • 属于类但是不属于对象
    • 静态方法不能在对象上调用，只能在类中调用

作用域

  • 作用域
  • 函数内声明的变量是局部变量，具有局部作用域
    • 函数内部的局部变量可以通过赋值给window对象的属性使其成为全局可访问的
  • 函数外声明的变量是全局变量，具有全局作用域
    • 全局作用域中的变量和函数是window对象的属性和方法，调用时省略window
  • 没有声明就使用的变量，默认为全局变量，不论这个变量在哪被使用

事件

  • HTML事件是发生在HTML元素上的事情
  • HTML事件可以是浏览器行为或用户行为
    • 事件触发时JS代码可以执行
  • HTML元素中可以添加事件属性
    • \
    • \

字符串

  • 引号中的字符为字符串
  • 可以使用索引位置来访问字符串中的每个字符
    • str[0]
  • 字符串长度
    • 内置属性 str.length
  • 特殊字符
    • 反斜杠转义字符
  • 字符串属性
    • 返回创建字符串属性的函数 constructor
    • 返回字符串的长度 length
    • 向对象添加属性和方法 prototype
  • 字符串方法

模板字符串

  • 在字符串中嵌入表达式和变量
  • 模板字符串是使用反引号(`)分隔的字面量
    • 允许多行字符串
    • 嵌入表达式的字符串插值
      • `string text ${expression} string text`
    • 带标签的模板的特殊结构
      • tagFunction`string text ${expression} string text`

运算符

  • 分类
    • 算术
    • 赋值
    • 关系
    • 逻辑
    • 条件
  • 优先级
  • 操作数

条件语句

  • if 语句 condition 为 true 执行 if 后的 code
    • if(condition){//code}
    • if(condition){//code}else{//code}
    • if(condition){//code}else if(condition){//code}else{//code}
  • switch
    • switch(x){case a://code break; case b://code break; default://code}
    • 使用 default 关键词来规定匹配不存在时做的事情
    • case的判断是数据类型和值的双重判断

循环语句

  • for 循环代码块一定的次数
    • for(initial; condition; after code){//code}
    • 遍历数组、字符串
  • for/in 循环遍历对象的属性
    • for(attribute in object){//code}
    • 遍历对象
  • while 指定条件为真时循环执行代码块
    • while(condition){//code}
  • do/while 检查条件是否为真之前执行一次代码块
    • do{}while(condition);
  • break 跳出循环
    • break label; 可以跳出 label 标记的语句代码块
  • continue 跳过当前迭代，继续循环下一个迭代

正则表达式

  • 使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式
  • 搜索模式可用于文本搜索和文本替换
  • 语法
    • /正则表达式主体/修饰符(可选)
  • 使用字符串方法
    • search(regex) 检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串，并返回子串的起始位置
    • replace(regex, str) 在字符串中用一些字符串替换另一些字符串，或替换一个与正则表达式匹配的子串
  • 模式
    • 方括号用于查找某个范围内的字符
      • [abc] 查找方括号之间的任何字符
      • [0-9] 查找任何从 0 至 9 的数字
      • (x|y) 查找任何以 | 分隔的选项
    • 元字符是拥有特殊含义的字符
      • \d 查找数字
      • \s 查找空白字符
      • \b 匹配单词边界
      • \uxxxx 查找以十六进制数 xxxx 规定的 Unicode 字符
    • 量词
      • n+ 匹配任何包含至少一个 n 的字符串
      • n* 匹配任何包含零个或多个 n 的字符串
      • n? 匹配任何包含零个或一个 n 的字符串
    • RegExp 对象
      • RegExp.test(str) 检测一个字符串是否匹配某个模式，如果字符串中含有匹配的文本，则返回 true，否则返回 false
      • RegExp.exec(str) 检索字符串中的正则表达式的匹配
  • 修饰符
    • i 执行对大小写不敏感的匹配
    • g 执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）
    • m 执行多行匹配

错误

  • throw exception
    • 创建或抛出异常
  • 语法
    • try{//在执行时进行错误测试的代码块，异常的抛出}catch(err){//异常的捕获与处理}finally{//结束处理，不论是否异常都会执行该代码块}

声明提升和初始化不提升

  • 函数及变量的声明都将被提升到函数的最顶部
  • 初始化不会提升到函数的最顶部
  • 在脚本或函数的开头使用 “use strict” 字面量表达式指令开启严格模式
    • 不允许使用未声明变量
    • 不允许删除变量或对象
    • 不允许删除函数
    • 不允许…

this关键字

  • 方法中 this 指向调用它所在方法的对象
  • 单独使用 this 指向全局变量
  • 函数中使用 this 默认表示全局变量([object Window])，严格模式下 this 是 undefined
  • 事件中 this 指向接收事件的 HTML 元素
  • 显式函数绑定，允许切换函数执行的上下文环境，即 this 绑定的对象
    • call(object)
    • apply(object)

let 和 const 关键字

  • let 实现在块级作用域中重定义和定义变量
  • const 声明常量，必须在声明时初始化

HTML DOM

简介

  • 文档对象模型 Document Object Model
  • HTML DOM 模型被构造为对象 Document 的树
    • JS可以改变HTML元素
    • JS可以改变HTML属性
    • JS可以改变CSS样式
    • JS能够对事件做出反应
  • 查找 HTML 元素
    • 通过 id
      • var x=document.getElementById(“idname”);
    • 通过标签名
      • var y=x.getElementsByTagName(“p”);
    • 通过类名
      • var x=document.getElementsByClassName(“classname”);

修改HTML元素的内容

  • 改变HTML输出流
    • document.write() 直接向HTML输出流写内容
    • 在DOM加载完成后使用document.write()会覆盖该文档
  • 改变元素的内容
    • document.getElementById(id).innerHTML=new content
    • innerHTML 属性表示元素的内容
  • 改变元素的属性
    • document.getElementById(id).attribute=new value
    • attribute 属性表示元素的属性
  • 过程
    • 获取元素
    • 通过 innerHTML attribute 属性修改元素内容

修改HTML元素的样式

  • document.getElementById(id).style.property=new value
  • style 属性表示样式，其中的 property:value 则是样式声明

对HTML DOM事件做出反应

  • 使用事件属性分配事件
    • 将以下声明加入元素的开始标签
      • 事件属性=”JS代码”
  • 使用 HTML DOM 分配事件
    • document.egtElementById(id).onclick=function(){displayDate()};
  • 事件属性/事件句柄
    • onload onunload 在用户进入或离开页面时被触发，可用于处理cookie
    • onchange 结合输入字段的验证来使用，改变输入字段时反应
    • onmouseover onmouseout 用于在用户的鼠标移至 HTML 元素上方或移出元素时触发函数
    • onmousedown onmouseup onclick 点击鼠标按钮时触发 释放鼠标按钮时触发 完成鼠标点击时触发
  • EventListener
    • element.addEventListener(event, function, useCapture);
    • 向指定元素添加事件句柄
    • 添加的事件句柄不会覆盖已存在的事件句柄
    • 可以向任何DOM对象添加事件监听
    • 使用”匿名函数”调用带参数的函数
      • element.addEventListener(“click”, function(){ myFunction(p1,
        p2); });
    • useCapture 默认为false
      • 事件冒泡 false
        • 内部元素的事件会先被触发，再触发外部元素
      • 事件捕获 true
        • 外部元素的事件会先被触发，再触发内部元素
    • element.removeEventListener(“mousemove”, myFunction); 移除事件句柄

添加或删除HTML DOM元素

  • 创建新的HTML元素
    • 添加新元素到尾部 appendChild()
      • 先创建一个元素
        • var para = document.createElement(“p”);
      • 创建新的文本结点
        • var node = document.createTextNode(“这是一个新的段落。”);
      • 在将文本结点添加到新元素中
        • para.appendChild(node);
      • 在已存在的元素中添加新元素
        • var element = document.getElementById(“div1”);
        • element.appendChild(para);
    • 添加新元素到开始位置 insertBefore(a, b)
      • 将a插入到b前面
  • 移除已存在的元素
    • 需要知道该元素的父元素
    • var child = document.getElementById(“p1”);
    • child.parentNode.removeChild(child);
  • 替换元素
    • parent.replaceChild(para, child);
    • 使用para替换child
  • Collection 对象
    • HTML元素的集合
    • getElementsByTagName(“p”) 返回 Collection 对象
    • 类似包含HTML元素的一个数组
  • NodeList 对象
    • 文档结点的集合
    • 类似 Collection 对象
