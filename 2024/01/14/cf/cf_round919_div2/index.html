Codeforces Round 919 Div2补题

C Partitioning the Array

problem: 给定n个数，根据n的因数分段，判断有多少个因数的分段满足同余意义下相等
solution: 最大公因数+同余性质。

a、b模m同余 <=> m|a-b。 m|a-b且p|m => p|a-b。

k满足存在m <=> k分段下的所有abs(a[i]-a[i+k])的最大公因数不为1

D Array Repetition

problem: 两种操作：末尾加数或重复数组，n次操作后q次查询位置k的数的值
solution:

寻找的位置，一定是操作1添加到末尾的数，操作二只会使得序列整体长度增加。
不直接记录修改后的数组，只记录操作，寻找的位置k必定位于序列长度大于等于k的最小的第i次操作

因此，可以通过递推式 k=k%num[i-1]不断缩小k，直到找到k是操作1添加到末尾的数，即k==num[i]或k%num[i-1]==0。
