动态规划

思想与实现

核心思想与相关定义

核心思想：从最简单的子问题开始，逐步扩大子问题的规模，最优解一定是从过去某一个子问题的最优解转移而来的

  • 决策
    • 会被决策影响的变量是状态特征
  • 状态
    • 维度的信息是离散的下标（取值范围），状态是维度组合表达的语义信息
    • 维度的值描述具体的状态，状态的值是最大/小值或方案数
  • DP方程
    • 动态规划本质上是一个DAG上的递推，按拓扑序求解保证无后效性
    • 状态转移方程可以通过列举所有决策并绘制状态转移图推出
    • 注意不同条件下的不同状态转移方程
  • 问题的时间复杂度=状态数目 乘 计算每个状态所需时间

基本思路

  1. 划分问题为若干阶段，每个阶段对应一个子问题（last状态），列举所有决策，对 last 状态的 last 操作可以帮助划分状态
  2. 在子问题中列举 last 状态的所有维度（会被决策影响且有取值范围的特征），将具有最优化单调性的值作为状态的值，其余值表示具体的状态
  3. 将状态根据决策条件分类，绘制状态转移图，推出全部决策条件下的状态转移方程
  4. 按拓扑序依次求解每一阶段问题（状态的值）

两种实现

  • 拓扑序递推：按拓扑序递推出结果
    • 我为人人：符合思考习惯
    • 人人为我：在选取最优备选状态的值（人人的和或人人中的最值）时可以降低时间复杂度
  • 记忆化搜索：记录已遍历的状态信息，避免重复遍历

应用

  • 最优化问题
  • 计数问题
  • 递推问题

背包DP

实现

01背包

  • 每个物品只有取与不取两种状态
  • 状态
    • 表示取到第 i 个物品时容量为 j 的背包获取的最大价值
  • 状态转移方程

  • 滚动数组优化第一维

完全背包

  • 每个物品可以取无限次
  • 状态同01背包
  • 状态转移方程
    • 正序遍历
  • 滚动数组优化第一维

多重背包

  • 每个物品有 个
  • 状态同01背包
  • 状态转移方程
    • 逆序枚举
  • 滚动数组优化第一维
  • 二进制分组优化
    • 将 进行二进制拆分，通过更少的拆分数量实现等效选择方式
  • 单调队列优化
    • 将对物品体积取模相同的背包当前容量进行分组，即按完全剩余系分组
    • 组内使用单调队列优化
      • 每次转移后使用单调队列维护

混合背包

  • 分别使用01背包、完全背包和多重背包解决

  • 泛化价值的背包，将固定价值换成函数调用

扩展

多维代价问题

  • 背包问题中代价是状态的维度
  • 状态开相应维度维护

分组背包

  • 多组物品，只能从一组中选择一件物品
  • 对每一组进行一次01背包，存储每组物品数量和编号

有依赖的背包

  • 选择一件物品必须选择其依赖的物品
  • 分组讨论，视为分组背包
  • 树形DP

泛化价值的背包

  • 将固定价值更换为函数调用

应用

  • 花费代价获取价值且存在代价上限的问题

树形DP

实现

树上背包

  • 选择子节点会花费父节点的代价并提供价值
  • 状态
    • 表示节点 i 子树下遍历前 j 个节点并花费代价 k 获取的最优解
  • 状态转移方程
  • 从叶到根的顺序作为DP的阶段
    • 先向下遍历一遍树，然后记忆化搜索记录每层状态的最优解，回溯时更新当前节点的最优解

换根 DP

  • 问题不指定根节点
  • 根节点的变化会对子节点深度和、点权和等信息产生影响
  • 两次 DFS
    • 第一次预处理会被根节点变化影响的信息
    • 第二次进行换根 DP
  • 求每个节点 k 距离范围内的点权和

优化

  • tarjan 缩点并设置超级源点建树

应用

  • 有依赖的背包
  • 树上最优化问题
  • 树上计数问题

数位DP

实现

  • 预处理最高 i 位为 j 的数出现的次数
    • 状态
      • 表示最高 i 位为 j 的数出现的次数
    • 状态转移方程
      • 第一个表示 i-1 位的贡献，第二个表示第 i 位的贡献
  • 预处理满 i 位每个数字出现的次数
    • 状态
      • 表示满 i 位每个数字出现的次数
    • 状态转移方程
  • 计数方法：
    • 将上界按位分开
    • 从高位到低位枚举，不贴着上界时低位随便取值，贴着上界时低位只能取到上界
    • 分两部分计算贡献(注意前导0)

思想

  • 数位是指把数字一位一位拆开，关注每一位上的数字
  • 计数问题中一个区间的答案可以拆成两个前缀和之差

应用

  • 统计满足一定条件的数的数量，条件转换后可用数位思想理解判断，输入给定数字区间限制统计范围
