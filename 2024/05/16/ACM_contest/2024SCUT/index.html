题意

给定一个初始全为空地 ‘.’ 的 大小的网格，可以往格子添加 ‘w’ 作为障碍，求使得网格任意两个空地之间均存在且只存在一条不经过障碍的简单路径的最少障碍放置方案。

分析

首先，对于一个网格，若其任意两个空地之间均存在且只存在一条不经过障碍的简单路径，则该图必定不存在一个长和宽同时大于1的区域块。对于大小为 的网格，加墙的数量最多为8，因此可以从 0 到 8 使用迭代加深搜索，构造满足条件的网格。
难点在于如何判断该网格是否满足任意两个空地之间均存在且只存在一条不经过障碍的简单路径，使用广搜形式的拓扑，对每个空地设置一个来源记录和访问记录，通过从初始点扩散，实现确定两个点之间的简单路径是否只有一条，若只有一条，则一个点必定只有一个来源且在扩散过程中不会从非来源点访问到该点。

题意

给一个 n 个点 n 条边的联通图，构造一个染色（红色或蓝色）方案使得图上的异色边有恰
好 k 条。

分析

n 个点 n 条边的联通图，是基环树。
首先通过 DFS 将唯一的环找到并标记，去掉环之后的森林中进行树上染色容易控制异色边数量，而环则不行。故先将环上每个点进行染色，且保证环上异色边不大于 k 的同时尽可能大，之后进行 DFS 对每棵树染色即可。
